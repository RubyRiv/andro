--------------------------------- Andro ------------------------------------

Der Pokemon Chatbot, der dich in deinen Kämpfen unterstützt 

----------------------------------------------------------------------------

Generation ist default-mäßig die neueste eingestellt, kann in Settings umgestellt 
werden, auch ausgestellt, dann wird jedes mal mit Slot-Filling nachgefragt, ansonsten
ist der Wert default-mäßig gesetzt. 

---------------------- 

Was ist sinnvoll als Bot und noch nicht vorhanden? 

Kämpfen? 

- Pokemon Stats anzeigen (damit man Gegner einschätzen kann) 
- Pokemon Schwächen anzeigen 
- What type is [Pokemon]? 
- [Pokemon] abilities
- [Type][Type] weaknesses
- Häufige Movesets (falls Smogon Daten teilt) 
- Erlernbare Attacken anzeigen 
	-> Bis bestimmtes Level 
	-> Von bestimmten Typ
	-> Physisch/Spezial 
	-> Ab bestimmter Stärke
 
- Can [Pokemon] hit [Pokemon]?
	-> (Sp.)Attack / (Sp.) Defense Vergleich 
	   Es wird gesagt, ob das gegnerische Pokemon mehr Attack oder Sp.Attack hat, danach 
	   wird das mit dem entsprechenden Verteidigungs-Wert des eigenen Pokemon verglichen. 

- Compare [Pokemon] and [Pokemon]
- Is [Pokemon] or [Pokemon] faster?
- Is [Pokemon] or [Pokemon] heavier? 

- [Pokemon] Defense [Attack] or [Attack] 
	-> Welche Attacke trifft das Pokemon härter? 

- [Pokemon] Attack [Attack] or [Attack]
	-> Mit welcher Attacke greift das Pokemon härter an? 

- [Attacke] Info 
- [Item] Info
- [Pokemon] STAB attacks 


Movesets? 
Quasi als Kampfvorbereitung. 

- Wich Pokemon learns [Attack][Attack][Attack][Attack]
	-> Find Pokemon which learn all four attacks 
- Wich Pokemon gets [abillity]
- How does [Pokemon] learn [Attack][Attack][Attack][Attack]
	-> Show when and how the Pokemon learns these attacks
- Counters to [Attack][Attack][Attack][Attack]
	-> Show which Pokemon can counter this moveset
- [Attack][Attack][Attack][Attack] typechart
	-> Show effectivness against types
- save ([Pokemon])[Attack][Attack][Attack][Attack] [name]
	-> saves set under given name to reuse later 

- Which Pokemon have abilities that go with [weather]? 
- double battle [moves]/[abillities]
- [Stat] raising attacks
